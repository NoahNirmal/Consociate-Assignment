# GitHub User Information Card

This project is a React application that allows users to enter a GitHub username and displays the public information of the GitHub user in a nicely designed card UI.

## Features

- Users can enter a GitHub username in a form.
- Upon submission, the application fetches the public information of the GitHub user using the GitHub API.
- The fetched user information is displayed in a beautifully designed card UI.
- The card includes the following information:
  - Avatar Image
  - Username
  - Name
  - Number of Public Repositories
  - Number of Public Gists
  - Profile Creation Date (in the format YYYY-MM-DD)

## Technologies Used

- React.js
- CSS (for styling)
- GitHub API
